## ✨ Concept

**Shakra** is not just another café website — it’s an **immersive digital experience**.  
Inspired by alchemy, midnight rituals, and sensory wonder, this front-end project redefines how a coffee brand can feel online. Every scroll, hover, and mouse movement responds with magic.

---

## 🎨 Unique Features (Wow-Factor)

| Feature | Description |
|---------|-------------|
| **Floating Particle Background** | 120+ glowing particles drift like star dust behind all content. |
| **Kinetic 3D Coffee Cup** | The cup rotates and tilts following your mouse cursor — alive and reactive. |
| **Custom Cursor** | A glowing coral orb that follows your every move, with mix-blend effect. |
| **Section-Based Alchemy** | Smooth page navigation with fade-slide transitions, each section named after mystical concepts (Essence, Alchemy Menu, Rituals, Conjure). |
| **3D Flip Menu Cards** | Menu items tilt in 3D on hover, creating a tactile, magical feel. |
| **Animated Counters** | Visitor stats (cups served, blends, rating) count up when you scroll into view. |
| **Ephemeral Steam Effect** | Rising steam particles from the cup image — pure visual delight. |
| **Floating Element Cards** | About section cards hover with a gentle, endless float animation. |
| **Fully Responsive** | Works on desktop, tablet, and mobile with a glassmorphism aesthetic. |

## 🧰 Tech Stack

- **HTML5** — semantic structure, canvas element.
- **CSS3** — custom properties, keyframe animations, backdrop‑filter, 3D transforms.
- **JavaScript (ES6)** — particle system, mouse tracking, Intersection Observer, dynamic menu rendering.
- **Font Awesome 6** — icons for atmosphere.
- **Google Fonts** — Space Grotesk + Cormorant Garamond.

## 🚀 How to Run Locally

1. **Clone or download** this repository.
2. Make sure the three core files are in the same folder:
   - `index.html`
   - `style.css`
   - `script.js`
3. Double-click `index.html` — it runs immediately in any modern browser (Chrome, Edge, Firefox, Safari).  
   *No build step, no server needed.*

> 💡 *For the best experience, use a desktop browser to see the custom cursor and kinetic cup effects.*
